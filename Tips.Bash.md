## Commands

- Show Command Help
  + ```help```
  + ```bash -c help```


- ```shopt```
  + https://www.gnu.org/software/bash/manual/html_node/The-Shopt-Builtin.html

  + show/change the settings of shell options
  + ```shopt -p``` - print each shell option with an indicator of whether it is set or not.
