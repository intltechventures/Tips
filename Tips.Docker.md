Windows Docker Service Commands
* Stop/Restart Docker Service
  * ```Net stop com.docker.service```
  * ```Net start com.docker.service```



Container Commands:
* List all containers
  * ```docker container ls --all```
* Remove a container
  * ```docker rm {container id}```


Image Commands:
* List docker images
  * ```docker image ls```
* Remove an image
  * ```docker image rm {image name}```
